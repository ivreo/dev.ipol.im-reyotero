SIFT Anatomy: Peer-reviewed implementation of the original SIFT method, published in IPOL.

SIFT Anatomy Extra: Implementation with additional functionalities for the numerical analysis of the SIFT method. Functionalities include the exact computation of the Gaussian scale-space, spline interpolation of the input image, EXR input and output, and various differential operators. This code is related to the journal article Analysis of the factors affecting keypoint stability in scale-space.
